LadinOS mirrors backed by [CDN Provider]

This is repo.ladinos.org. This service provides mirrors for the following LadinOS archive repositories:

  • /ladinos/
  • /ladinos-debug/
  • /ladinos-experimental/

The server repo.ladinos.org does not host packages directly, but the name has SRV records in DNS that allow SAPT (LadinOS’s official package manager) in LadinOS 1.0 and later to automatically discover optimal mirror locations.

Using with SAPT (via SRV records)

To use this service with a sufficiently recent SAPT, add the following lines to your sources.list (typically located at /etc/sapt/sources.list):

sapt repo http://repo.ladinos.org/ladinos stable main
sapt repo http://repo.ladinos.org/ladinos-debug stable-debug main
sapt repo http://repo.ladinos.org/ladinos-experimental unstable main

SRV Record Details

As of 2025-09-13, the DNS SRV record for repo.ladinos.org is:

_http._tcp.repo.ladinos.org.    IN      SRV     10 1 80 cdn.ladinos.mirror.cdnprovider.net.

Direct CDN Access (No Redirects)

If you use an older SAPT version (that does not support SRV records) or a network environment with limited DNS functionality, requests to repo.ladinos.org will be HTTP redirected to CDN instances. To avoid redirects, use the CDN endpoint directly in sources.list:

sapt repo http://cdn.ladinos.org/ladinos stable main
sapt repo http://cdn.ladinos.org/ladinos-debug stable-debug main
sapt repo http://cdn.ladinos.org/ladinos-experimental unstable main
